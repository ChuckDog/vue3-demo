# vue3-demo

vue3入门教程

包含`响应式API`、`组合式API`、`其它特性`、`Counter` 以及 `TodoMVC`，
同时集成 `vuex4`、`vue-router4`、`typescript`和`vitejs`。

关于vue3整体上的变化总结，可以阅读这篇博客[《vue3编程正确姿势》](https://zhuanlan.zhihu.com/p/328694623)。
